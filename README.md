# BarterBite

BarterBite is a *React web app* that allows users to trade food items directly with others. Users can open personal stores, list items they want to trade, and connect with a community to exchange food â€” no cash required.

## Features

- Create a personal store/profile
- List food items with descriptions, photos, and quantities
- Search and match with users who have what you need
- Trade directly or via in-app trade credits
- Rate and review other users for trust and safety

## Tech Stack

- Frontend: React (Create React App)
- Backend: None (static React app deployed on Vercel)
- Deployment: Vercel (free hosting)
